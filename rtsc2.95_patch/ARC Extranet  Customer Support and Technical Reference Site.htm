The information in this document applies to:
• RTCS 2.95

This patch (RTCS2.95_Patch04) can be used to upgrade RTCS 2.95 to incorporate all the latest bug fixes and feature enhancements available from ARC International. Please only apply this patch if you are using RTCS 2.95 and you cannot upgrade to RTCS 2.96 (which incorporates all these fixes and quite a few other ones). If you are using an earlier version of RTCS and wish to upgrade to the latest version available for your product please contact swupgrades@arc.com using the following form.
